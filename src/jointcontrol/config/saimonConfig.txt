The JSON File describes each joint configuration whose controller parameters shall be defined by the model.

JSON Entries:

NumJoints: number of defined joints
Joints: joint data

Name: name of current joint
NumParams: number of settable parameters for this joint
Defaults: default parameters for this joint (len of array must match NumParams)
Minimums: minimum for each parameter that can be set (len of array must match NumParams)
Maximums: maximum for each parameter that can be set (len of array must match NumParams)
ParamDescription: description what each parameter represents

ROS Parameter Setup:



Parameter structure:

/camera/left/name: leftcamera
/camera/left/exposure: 1
/camera/right/name: rightcamera
/camera/right/exposure: 1.1
